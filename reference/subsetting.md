# Subsetting `ir` objects

Subsetting `ir` objects

## Usage

``` r
# S3 method for class 'ir'
x[i, j, ..., exact = TRUE]

# S3 method for class 'ir'
x$i

# S3 method for class 'ir'
x[[i, j, ..., exact = TRUE]]

# S3 method for class 'ir'
`$`(x, i, j, ...) <- value

# S3 method for class 'ir'
i[j, ..., exact = TRUE] <- value

# S3 method for class 'ir'
i[[j, ..., exact = TRUE]] <- value
```

## Arguments

- x:

  An object of class `ir`.

- i, j:

  Row and column indices. If `j` is omitted, `i` is used as column
  index.

- ...:

  Ignored.

- exact:

  Ignored, with a warning.

- value:

  A value to store in a row, column, range or cell. Tibbles are stricter
  than data frames in what is accepted here.

## Value

If the subsetting operation preserves a valid `spectra` column (see
[`ir()`](https://henningte.github.io/ir/reference/ir_new_ir.md)), an
object of class `ir` with accordingly subsetted rows or columns. Else a
[`tibble::tbl_df()`](https://tibble.tidyverse.org/reference/tbl_df-class.html)
or vector.
